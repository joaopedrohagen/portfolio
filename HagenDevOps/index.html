Tecnologias utilizadas nesse site:

HMTL5

HTML5 é uma linguagem de marcação utilizada para desenvolver páginas web. É a quinta versão do HTML e foi lançada em 2014. O HTML5 oferece novas funcionalidades, incluindo suporte a vídeo e áudio nativos, novas tags semânticas, elementos de formulário melhorados e a capacidade de armazenar dados localmente no navegador. Isso permite a criação de aplicativos web mais avançados e conteúdo interativo, tornando a web mais acessível e interativa. 

CSS3

CSS3 é uma linguagem de estilo utilizada para estilizar páginas web. É a terceira versão de CSS e oferece novas funcionalidades como bordas arredondadas, sombreamento de texto, gradients, animações e media queries. O CSS3 permite aos desenvolvedores controlar a aparência de elementos na página, tornando mais fácil criar designs elegantes e responsivos que se adaptam a diferentes tamanhos de tela. Em resumo, CSS3 é fundamental para o desenvolvimento de páginas web modernas e atraentes. 

JavaScript é uma linguagem de programação de alto nível, dinâmica e interpretada. Foi criada em 1995 com o objetivo de tornar a web mais interativa e dinâmica. Atualmente, ela é amplamente utilizada para desenvolvimento de aplicações web, desde simples páginas dinâmicas até aplicações complexas e interativas. Além disso, o JavaScript também pode ser executado em servidores, dispositivos móveis e na nuvem, tornando-o uma linguagem de programação versátil e altamente procurada no mercado de trabalho. Seu código é fácil de aprender e seus recursos avançados permitem criar soluções sofisticadas e inovadoras para a web.

Nginx

NGINX é um software de servidor web e proxy reverso utilizado para lidar com altos volumes de tráfego na Internet. Ele oferece recursos avançados, como balanceamento de carga, roteamento dinâmico de requisições e suporte a protocolos de segurança, além de ser conhecido por sua facilidade de administração. É uma escolha popular para equipes de desenvolvimento e operações que precisam garantir uma experiência rápida e segura para os usuários da web. 

Docker

Docker é uma plataforma de virtualização de aplicações que permite a execução de aplicações em contêineres autônomos. Isso facilita a colaboração e o compartilhamento de aplicações entre equipes de desenvolvimento, além de tornar a implantação e gestão de aplicações mais eficiente. É amplamente utilizado em nuvem, devOps e microserviços para entrega ágil e escalável de aplicações. 

Kubernetes

Kubernetes é uma plataforma de gerenciamento de contêineres que automatiza a implantação, escalabilidade e gestão de aplicações em contêineres. É amplamente utilizado em nuvem e empresas, tornando a entrega de aplicações mais eficiente e escalável. Ele é altamente personalizável e pode ser integrado com outras ferramentas.
